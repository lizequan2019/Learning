有时开发是需要指定python版本的,所以就不要使用包管理器下载python了,直接去官网下载,选择对应版本的python源码,我使用的是3.9.5

下载后编译,执行 ./configure --prefix="安装目录"就行了

然后就会在安装目录下生成一些文件,如下

            bin  include  lib  pylib_3party  share

接着设置永久环境变量,这样你调用python命令时就不用加路径了,在~/.bashrc文件下增加如下语句

            export PATH="$PATH:安装目录"

python的优势之一就是拥有丰富的第三方库,所以我们会使用pip下载库,默认是下载到一个位置,我们需要在下载完成后将文件移动到自己安装目录下的pylib_3party目录

还要在~/.bashrc文件下增加如下语句

            PYTHONPATH是一个环境变量,用于在Python中添加额外的库搜索路径

            export PYTHONPATH="$PYTHONPATH:/home/lzq/build-for-python/pylib_3party"

这样python程序就能找到下载的库了
