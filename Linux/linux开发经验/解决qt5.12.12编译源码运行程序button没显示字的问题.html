编译前会执行configure操作,注意输出结果中的这几行

            Required system libraries:
            fontconfig ........................... yes  <============ 这个
            dbus ................................. yes
            nss .................................. no
            khr .................................. yes
            glibc ................................ yes

这里确保fontconfig是yes,如果没有的话就安装一下,如下

            sudo apt-get install libfontconfig1-dev
