一维数组指针(也称行指针),可以理解为指向一维数组空间的指针。

下面是数组指针的两种写法

int(*p)[n]; //如果*p不加括号,会变成 int * p[n] 指针数组 int[n] *p; //这是另一种写法，感觉更好理解

为什么*p和[n]变换位置就需要加一个括号罩住*p,这是因为运算符优先级的问题，[] = () > *, (*p) 表示 *p 是一个整体表示一个指针。感觉和函数指针一样

指针数组(指针类型数组)

int *p[n] ===> int *(p[n])

这个可以类比char a[n](char类型数组),int b[n](int类型数组),那么int *p[n]自然就是int指针类型数组
