### 使用方法

```c++

    #include <iostream>

    using namespace std;

    //这种格式在调用时可以增加额外的输出内容 format 就是额外输出的
    #define debug(format,...) printf("%s "  format  "\n",__FILE__,__VA_ARGS__);

    //这种格式只能输出已经规定好的内容 这里只能输出 %s %d %d 
    #define debug2(...)       printf("%s %d %d \n",__FILE__,__VA_ARGS__);

    int main(){

        int a = 10;
        char * ch = "i am linux";
        debug("%d  %s",a,ch);

        debug2(10,11);

        return 0;
    }

/*
执行结果
    test.cpp 10  i am linux
    test.cpp 10 11
*/
```

&emsp;&emsp;输出debug字符可以变颜色的操作(win和linux都可使用，win可能使用部分)

```c++
    #define debug(format,...) 
    printf("\e[47;31m LINE = %d  FUN = %s\"  format "\e[0m\n",__LINE__,__FUNCTION__,__VA_ARGS__);

    对参数的解释 \e[F;B;Om
    \e 转义字符表示开始
    [ 开始定义颜色。 
    'F' 为字体颜色，编号30~37；'B' 为背景色，编号40~47。
    'O' 为特殊意义代码，它们的顺序没有关系。可以为 0 1 4 5 7 8
    m 是标记
  


    前景           背景           颜色
    ---------------------------------------
    30             40            黑色
    31             41            红色
    32             42            绿色
    33             43            黄色
    34             44            蓝色
    35             45            洋红
    36             46            青色
    37             47            白色

    特别代码意义
    -------------------------
    0 OFF
    1 高亮显示
    4 underline
    5 闪烁
    7 反白显示
    8 不可见
```
